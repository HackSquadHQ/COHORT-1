Networking Basics
 Wireless router acts as DHCP server for all local hosts attached to it either by Ethernet cable or wirelessly 
  DHCP means dynamic host configuration protocol 
The process used to convert private addresses to internet-routable addresses is called NAT
  Data originated with an end device, flows through the network and arrives at an end device.
 Address resolution protocol (ARP) is used to determine the device MAC address of a known destination device IPv4 address.
Neighbour discovery (ND) is used to determine the MAC address of a known destination device IPv6 address 
UDP is a best effort delivery system that does not require acknowledgment of receipt
